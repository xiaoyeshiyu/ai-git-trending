- **可以用在哪些场景**：替代 Node + npm 的本地开发环境，直接运行 JavaScript/TypeScript 项目并管理依赖；在前端项目里承担依赖安装、测试执行和打包构建，减少工具拼装；在 CI/CD 流水线中缩短安装依赖和执行测试的时间，适合构建频繁的仓库。
